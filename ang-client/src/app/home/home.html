EthioTrade

Where you can search Ethiopian import and export data.

The purpose of this site is to make Ethiopian import and export statistics available in a userfriendly and clear format in order to promote:

  • Investment

    assist potential investors discover opportunities for import substitution.

  • Growth

    help businesses identify developing trends to guide decisions.

  • Research

    serve as a reference for students, researchers and decision makers.

You can search and select an HS Code from the table, select a country from the drop down in the header or check out one of the annual summaries from the dropdown link above. Good luck and happy browsing!

The data on this site is available to the public from the Ethiopian Revenue and Customs Authority (ERCA) and is updated every two months.

This data represents the total annual import and export volume in USD since 1997. The data was aggregated from the Ethiopian Revenue and Customs Authority.

For a more detailed summary of the annual trade volume, you can select the year from the dropdown in the navigation bar above.
